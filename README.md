## **Overview**
The **To-Do Manager** is a lightweight task management application that helps users stay organized by allowing them to **add, mark, and delete** tasks efficiently. It leverages **localStorage** to save tasks so that they persist even after refreshing the page.

---

## **Features ✅**
- 📝 **Add New Tasks** – Enter a task and add it to your list.  
- ✅ **Mark Tasks as Completed** – Click the checkbox to mark a task as done.  
- 🗑️ **Delete Tasks** – Remove tasks easily with a delete button.  
- 💾 **LocalStorage Support** – Tasks remain saved even after page reload.  
- 🎨 **Minimal and Responsive UI** – Built with Bootstrap for a clean look.  

---

## **Technologies Used 🛠️**
- **HTML** – Page structure  
- **CSS** – Styling and layout  
- **Bootstrap 4** – Responsive design  
- **JavaScript (Vanilla JS)** – Logic for adding, deleting, and marking tasks  
- **localStorage** – Data persistence  

---

## **How to Use? 📌**
1. **Visit the live site:** ([https://your-todo-app-url.com/](https://gettasksdone.ccbp.tech/))  
2. **Type a task** into the input field and click "Add" or press "Enter".  
3. **Check the checkbox** to mark a task as completed.  
4. **Click the trash icon** 🗑️ to delete a task from the list.  
5. **Your tasks are saved automatically** – Close the browser and come back anytime!
